README: Linear Regression (Lab Question)

This code implements Linear Regression on the given dataset (Q1.xlsx) 
using two methods:
1. Closed-Form Solution (Normal Equation)
2. Gradient Descent (Custom Implementation)

-----------------------------------------
Requirements:
-----------------------------------------
Python 3.x with:
- numpy
- pandas
- matplotlib
- openpyxl (for reading Excel files)

Install (if needed):
    pip install numpy pandas matplotlib openpyxl

-----------------------------------------
How to Run (Method 1: Python Script)
-----------------------------------------
1. Use the exported Python file (Q1.py).
2. Keep Q1.xlsx in the same folder.
3. Run in terminal / command prompt:
       python Q1.py

-----------------------------------------
How to Run (Method 2: Jupyter Notebook)
-----------------------------------------
1. Open the notebook file (Q1.ipynb) in Jupyter:
       jupyter notebook
2. Run the cells in order.

-----------------------------------------
Outputs:
-----------------------------------------
- Scatter plots of features vs Y
- Training and validation error plots
- Errors for train, validation, and test sets
